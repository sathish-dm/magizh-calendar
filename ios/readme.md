# Magizh (மகிழ்) - Tamil Calendar

Modern Tamil calendar app with location-aware Panchangam calculations and smart food planning.

## Tech Stack
- Swift 5.9+
- SwiftUI
- iOS 16+ (targeting iOS 18 design language)
- MVVM Architecture

## Design Language
iOS 18 Liquid Glass - translucent materials, soft glows, depth

## Key Features
1. Location-aware Panchangam calculations
2. Smart food planning alerts (killer feature)
3. Traditional five-angam structure
4. Beautiful liquid glass UI
